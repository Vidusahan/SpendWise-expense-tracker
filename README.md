# SpendWise — AI-Powered Personal Expense Management Platform

## 📌 Project Overview
SpendWise is a secure, scalable personal finance management application. It helps users track expenses, manage budgets, and receive intelligent financial insights to understand their spending habits better.

## 🚀 Tech Stack
* **Frontend:** React, TypeScript, Tailwind CSS, React Router, React Query
* **Backend:** Node.js, Express.js, TypeScript, Prisma ORM
* **Database:** PostgreSQL
* **Infrastructure:** Docker, GitHub Actions (CI/CD)
